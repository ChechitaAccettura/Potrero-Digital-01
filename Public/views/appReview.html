Caravanya
Caravanya no es otra app para encontrar campings, es LA app. Campings, normales o para autocaravanas, lugares para acampar y aparcar libremente, todo con información detallada y fotos ¡en nueve idiomas diferentes!

iOverLander
iOverlander es una base de datos de lugares para overlanders y viajeros. Incluye campamentos, hoteles, restaurantes, mecánica, agua, llenado de propano y otras categorías. Los detalles se enumeran para cada lugar, incluidos los servicios, las fotos, la fecha de la última visita y las coordenadas del GPS.
